# 🌿 Dashboard Fauna Silvestre (v4.0 - Visual Mejorado)

Dashboard interactivo para análisis de inventarios forestales con **accesibilidad visual mejorada** y gráficos de alta legibilidad.

## 🎨 Mejoras Visuales Aplicadas
- ✅ **Tipografía oscura**: Todos los títulos y etiquetas en `#000000` con tamaño 1.1rem+ para alta legibilidad.
- ✅ **Colores categóricos**: Cada categoría (CUA, Clase, Ecozona, Departamento) tiene color distintivo usando paleta `Bold`.
- ✅ **Top 10 Especies**: Cada barra con color único para máxima distinción visual.
- ✅ **Composición**: Reemplazado Treemap por **barras agrupadas** más legibles.
- ✅ **Interfaz limpia**: Eliminado elemento "Espacio adicional" innecesario.
- ✅ **Bordes definidos**: Contenedores con borde `#2C3E50` para separación visual clara.

## 🚀 Características
- 🔍 Filtros independientes (UM_id, Departamento, Ecozona, Año, Clase, Familia, CUA)
- 📊 6 pestañas de visualización interactiva con Plotly
- 📦 Exportación completa de datos filtrados a CSV
- ♿ Accesibilidad visual para usuarios con baja visión
