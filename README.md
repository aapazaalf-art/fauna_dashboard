# 🌿 Dashboard Fauna Silvestre

Dashboard interactivo para análisis de inventarios forestales.

## 🚀 Características
- 🔍 Filtros independientes (UM_id, Departamento, Ecozona, Año, Clase, Familia, CUA)
- 📊 Visualizaciones interactivas con Plotly
- 📦 Exportación completa de datos filtrados a CSV
- ♿ Accesibilidad visual mejorada (texto oscuro, bordes definidos)
