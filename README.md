# 🌿 Dashboard Explorador de Fauna Silvestre (v2.0 - Producción)

Dashboard interactivo para el análisis de inventarios forestales nacionales. Permite filtrar por Unidad Muestral, Departamento, Ecozona, Año, Clase, Familia y CUA, con visualización de riqueza taxonómica y exportación completa de datos.

## 🚀 Características Clave
- 🔍 **Filtros Independientes**: La selección en una dimensión NO bloquea las opciones de las demás.
- 🗺️ **Riqueza por Ecozona**: Gráfico específico con X=Ecozona, Y=Espp. Únicas.
- 🌳 **Treemap Mejorado**: Paleta de colores vivos y tipografía oscura para lectura inmediata.
- 📊 **Análisis CUA**: Visualización clara de especies por Clasificación de Uso Actual.
- 📦 **Exportación Exacta**: Descarga masiva de **todas las filas y columnas filtradas**, sin paginación ni recortes.
